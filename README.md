# Time Counter API

Mocked API build with [`json-server`](https://github.com/typicode/json-server).
View the [online demo](https://time-counter-api.herokuapp.com/) deploy with [Heroku](https://dashboard.heroku.com).

### `GET /leaderboard`

### `POST /leaderboard`

Headers:

  "score": 999
